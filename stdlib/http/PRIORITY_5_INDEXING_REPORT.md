## 🛠️ B-tree 구현 내용

### 알고리즘 특성

```
B-tree (Degree = 32):

선형 검색:
for record in records:
  if record.id == target:
    return record
시간복잡도: O(n)

B-tree 검색:
node = root
while not found:
  index = binary_search(node.keys, target)
  if found: return node.values[index]
  if leaf: return null
  node = node.children[index]
시간복잡도: O(log n)
```
